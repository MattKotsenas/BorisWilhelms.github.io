Disable Application Insights Correlation Ids on HttpClient requests in ASP.NET Core

2017-05-26

With the upcoming version (2.1) of the Application Insights package for ASP.NET Core all requests made with HttpClient (and probably other clients) will have addition request headers, contain correlation ids. These headers are Request-Context, Request-Id, x-ms-request-id and x-ms-request-root-id.
